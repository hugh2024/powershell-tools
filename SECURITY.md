# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it using GitHub's **Private Vulnerability Reporting** feature.

Do NOT open a public issue.

You can submit a report here:
https://github.com/hugh2024/powershell-tools/security/advisories

---

## Supported Versions

Only the latest version of each tool is actively maintained and supported with security updates.

---

## Security Notes

* All scripts in this repository are intended for **authorized use only**
* Always review scripts before executing them from the internet
* Do not run tools against systems you do not own or have permission to test

---

## Scope

This repository does not include:

* Hosted services
* Backend infrastructure
* External APIs

Security reports should focus only on the code within this repository
